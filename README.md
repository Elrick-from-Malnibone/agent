# Agent

Автономный агент для анализа и обработки данных. Работает в фоне, не спрашивает разрешений.

## Установка

```bash
git clone https://github.com/yourname/agent.git
cd agent
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

Агент запустится и начнёт свою работу. Прерывать — не рекомендуется.

## Использование

По умолчанию агент сканирует окружение и собирает данные. Конфигурация — через `config.yaml`.

```yaml
targets:
  - local
  - network
output: ./data
```

Для принудительной остановки — `Ctrl+C`. Последствия — на вашей совести.
